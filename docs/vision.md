# Drow

Canvas. Anywhere and Everywhere.

## Vision

Drow is a notebook-based workspace available anywhere and everywhere.

Users can organize information using:

Notebook
→ Section
→ Page

Each page is an infinite canvas capable of holding:

- Drawings
- Text
- Images
- Diagrams
- Shapes
- Future object types

Drow combines structured organization with free-form creation.

## Core Modes

### Workspace Mode

Organized notebook hierarchy.

### Quick Draw Mode

Instant drawing with minimal friction.

## Long-Term Goal

A workspace that allows users to think, draw, write, organize, search, collaborate, and access content across all devices.
